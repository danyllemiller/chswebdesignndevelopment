Chapter 6: Tech Talk: Your A-Z Glossary

Master the language of UI/UX with these core terms.

Essential Terminology

Accessibility (a11y)
Designing so people with disabilities can use your site.
Breadcrumbs
A secondary navigation scheme that reveals the user's location.
Card
A UI pattern that groups related info (image, headline, button) into a container.
Cognitive Load
The amount of mental effort required to use an interface. Keep it low!
Contrast
The difference in brightness between text and background. High contrast = readable.
Figma
A web-based vector graphics editor and prototyping tool.
Hamburger Menu
A toggle menu for hiding navigation on mobile devices.
Hero Section
The introductory section of a website, usually containing a key image and message.
High-Fidelity
A design that looks exactly like the final product.
Jakob's Law
Users prefer your site to work the same way as all the other sites they already know.
Mobile-First
A design philosophy that starts with the mobile viewport and scales up.
Mockup
A static design representation used to demonstrate the look of the site.
Modal
A pop-up window that appears on top of the page, blocking other interactions until it is closed.
UI (User Interface)
The graphical layout of an application.
UI Pattern
A reusable solution to a common design problem (e.g., Cards, Modals, Breadcrumbs).
Usability Testing
The process of observing real users trying to complete tasks on your site to find design flaws.
UX (User Experience)
How a user interacts with and experiences a product.
Wireframe
A basic, skeletal visual guide used in interface design.
