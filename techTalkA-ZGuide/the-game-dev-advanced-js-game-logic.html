Tech Talk: Chapter 12

Exhaustive Vocabulary for Game Dev, Physics & OOP

AABB Collision
Axis-Aligned Bounding Box; the most common method for detecting rectangle collisions by checking the overlap of x, y, width, and height.
Acceleration
The rate at which an object's velocity increases over time. In games, this is used to make movement feel more natural.
Asset Loader
Code designed to ensure all images and sounds are 100% downloaded before the game loop starts to prevent errors or "invisible" objects.
cancelAnimationFrame()
The command used to instantly stop a game loop that was triggered by requestAnimationFrame.
Example: cancelAnimationFrame(gameLoopId);
Canvas API
A high-performance HTML5 system for drawing 2D graphics and sprites directly in the browser via JavaScript.
Circular Collision
Collision detection that checks if the distance between the center points of two circles is less than the sum of their radii.
constructor()
The specialized function inside an ES6 Class that executes automatically every time a brand new object (instance) is created.
Example: constructor(x, y) { this.x = x; }
ctx.arc()
The Canvas method used to draw curves or full circles. It requires a starting and ending angle in radians.
Example: ctx.arc(x, y, radius, 0, Math.PI * 2);
ctx.beginPath()
Starts a new drawing path; essential to call before a new shape to prevent it from connecting to previous shapes.
Example: ctx.beginPath(); ctx.moveTo(0,0);
ctx.clearRect()
Erases a specific area of the canvas. This is the first step of a professional game loop to prevent "smearing."
Example: ctx.clearRect(0, 0, canvas.width, canvas.height);
ctx.drawImage()
The primary method used to render external image files (sprites) onto the game board.
Example: ctx.drawImage(heroImage, heroX, heroY);
ctx.fillStyle / strokeStyle
Properties that set the interior color (fill) or border color (stroke) of shapes before they are drawn.
Example: ctx.fillStyle = "#FF0000";
ctx.save() / restore()
A pair of commands that saves the current state of the canvas (colors, rotation) and then restores it, preventing one object's rotation from affecting the next.
Example: ctx.save(); ctx.rotate(1); ... ctx.restore();
Delta Time
The time elapsed since the last frame. Used to normalize movement speed so Gabe has the same experience on a 60Hz or 144Hz monitor.
Encapsulation
The OOP principle of grouping all related properties and methods inside a single Class blueprint for organization.
event.code
The property of a keyboard event that identifies the physical key pressed (e.g., 'Space', 'ArrowUp'), ignoring the character itself.
Example: if (event.code === "ArrowRight") { ... }
extends / super()
Keywords used for Class inheritance. `extends` creates a subclass, and `super()` calls the parent's constructor.
Example: class Dog extends Animal { constructor() { super(); } }
Friction
Simulated physics where velocity is multiplied by a value less than 1 (e.g., 0.98) every frame to gradually slow an object down.
Game Loop Order (C.U.D.)
The professional execution sequence: Clear the Canvas → Update logic/physics → Draw the objects.
Game State
A variable (often a string or boolean) that controls which logic blocks are currently active (e.g., 'menu', 'playing', 'gameover').
Gravity
Simulated by adding a constant, positive value to an object's vertical velocity (Y-axis) every single frame.
Haptic Feedback
Physical communication through touch or vibration, often used to alert Gamer Gabe to a hit or a win.
Hitbox
The invisible area (usually a rectangle or circle) around a sprite used to calculate if a collision has occurred.
Instantiate (new)
The process of creating a brand new, active object from a Class blueprint using the `new` keyword.
Example: const boss = new Enemy();
Magic Numbers
Hard-coded, unexplained numbers in your code (e.g., 800) that should be replaced with named constants (e.g., CANVAS_WIDTH).
Parallax Scrolling
Moving background layers at different speeds to create a 3D illusion of depth on a 2D screen.
Particle Systems
A technique for creating complex effects like fire, smoke, or explosions using dozens of small, short-lived objects.
Pythagorean Theorem
The math formula ($a^2+b^2=c^2$) used to find the distance between two center points for circular collision detection.
requestAnimationFrame()
The pro standard for game loops; it syncs with the screen's refresh rate and pauses automatically when the tab is hidden.
Example: requestAnimationFrame(gameLoop);
shift() / splice()
Array methods used for cleaning. `shift()` removes the first item; `splice()` removes an item at a specific index.
Example: enemies.splice(hitIndex, 1);
Sprite Sheet
A single image file containing every frame of an animation sequence to save memory and loading time.
this
A keyword referring to the specific object instance currently being updated or drawn inside a Class.
Example: this.health -= 10;
Velocity
A vector value representing both the speed and the direction of an object's movement.
