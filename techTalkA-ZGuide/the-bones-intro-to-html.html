Tech Talk: Chapter 7

Exhaustive Vocabulary for HTML Structure & Semantics

Absolute Path
A file path that starts from the website's root folder or includes a full URL (e.g., https://google.com or /images/logo.png).
Accessibility (a11y)
The practice of designing web pages to be usable by everyone, including people with disabilities (e.g., using alt text for screen readers).
Attribute
Extra information inside an opening tag that changes its behavior (e.g., href, src, id, class).
Boilerplate
The standard, "starter" code you put in every HTML file (<!DOCTYPE html>, <html>, <head>, <body>).
Character Entity
A code for displaying special symbols that aren't on the keyboard, like &copy; for © or &nbsp; for a space.
CSS (Cascading Style Sheets)
The language used to style your HTML, controlling colors, fonts, and layout.
Element
The entire HTML unit: opening tag + content + closing tag.
Empty Element
A tag that does not have a closing tag because it doesn't wrap content (e.g., <img>, <br>, <hr>).
<footer>
The semantic tag for the closing content of a page or section (often contains copyright or contact info).
<header>
The semantic tag for the introductory content of a page or section (often contains a logo and site title).
href
The attribute that specifies the "hyperlink reference," or destination URL, for an <a> tag.
<link>
A tag used in the <head> to connect external files, most commonly a CSS stylesheet.
<main>
The semantic tag that contains the main, unique, and dominant content of a page.
<meta> tags
Tags in the <head> that provide invisible metadata, such as charset="UTF-8" and name="viewport".
<nav>
The semantic tag used to define a block of navigation links.
Nesting
Placing one HTML element completely inside another (like a <li> inside a <ul>).
Relative Path
A file path expressed in relation to the current file (e.g., about.html or images/logo.png).
Responsive Design
The practice of making a website's layout adapt gracefully to look good on all devices (phones, tablets, desktops).
Semantic HTML
Using HTML tags that clearly describe their purpose (like <article>) instead of generic tags (like <div>).
src
The attribute that specifies the "source" file for an embedded element, like an <img> tag.
Text Editor
A specialized program for writing code (like Visual Studio Code or Notepad++).
URL (Uniform Resource Locator)
A standard, globally unique address for a resource on the web.
