Tech Talk: Chapter 18

Exhaustive Vocabulary for Deployment & Professional Portfolios

Absolute Path
A file path that references the full directory of a local computer (e.g., C:/Users/Hannah/Desktop...). These break immediately upon deployment because servers do not share your local folder structure.
A Record (Address Record)
The most basic type of DNS record. it maps a domain name directly to the numeric IP Address of the hosting server.
Accessibility Audit
A QA process (often using Lighthouse) to ensure a site follows WCAG standards, checking for Alt Text, high color contrast, and keyboard operability.
Case Study
A detailed narrative in a portfolio that explains the "story" of a project, highlighting the problem solved, the tools used, and the final results.
CDN (Content Delivery Network)
A global network of servers (like the one Netlify uses) that stores copies of your website assets. It delivers content to users from the server physically closest to them, reducing lag.
CI/CD (Continuous Integration / Continuous Deployment)
A modern, automated workflow where code is frequently merged into a hub (GitHub) and then automatically deployed to a live server without manual file uploads.
CNAME Record (Canonical Name)
A DNS record that maps one domain name to another (aliasing). For example, pointing www.hannahcodes.com to hannah-portfolio.netlify.app.
Deployment
The high-stakes process of moving code from a local, private development environment into a public, live Production environment.
DNS (Domain Name System)
The internet's master "phonebook." It translates human-friendly URLs (like google.com) into the numeric IP addresses computers use to communicate.
DNS Propagation
The time (up to 24 hours) it takes for a change in DNS settings to spread to every internet provider and server across the globe.
Favicon
The small "Favorite Icon" displayed in the browser tab next to the page title. It is the final professional touch for brand consistency.
FileZilla
A famous, open-source software program used as a digital "moving truck" to transfer files via the legacy FTP protocol.
FTP (File Transfer Protocol)
The legacy set of rules for moving files over the internet. It is manual, prone to human error, and lacks the safety of version control.
Google Lighthouse
An automated, professional auditing tool built into Chrome. It grades sites from 0-100 on Performance, Accessibility, Best Practices, and SEO.
Imposter Syndrome
The common psychological feeling that you aren't a "real" professional or that your skills aren't good enough, despite your actual accomplishments.
index.html
The "Index Rule" file name. Almost all web servers automatically recognize this specific file as the default homepage for a directory.
IP Address
The unique numeric identifier for every device on a network (e.g., 192.168.1.1). DNS translates these into human-readable words.
Link Rot
A situation where hyperlinks on a site no longer work because the destination pages or files were moved or deleted, resulting in 404 Errors.
PDF Resume
The professional standard for resumes in portfolios. They are preferred because they are easy to print and scan into Applicant Tracking Systems (ATS).
Production Environment
The final "Live" state of a website where real users interact with it. This is the opposite of the Local or Development environment.
QA (Quality Assurance)
The final process of testing a site for bugs, broken links, and errors. It includes the Resize Test to verify mobile responsiveness.
Registrar
A company (like Namecheap or GoDaddy) authorized to sell and manage domain name registrations.
Relative Path
A file path defined relative to the current file (e.g., images/logo.png). This is the mandatory standard for professional deployment.
SSL/TLS Certificate
The technology that enables the padlock icon and HTTPS, ensuring that all data sent between the user and server is encrypted.
STAR Method
A logical structure for portfolio descriptions: Situation, Task, Action, Result.
Subdomain
A prefix added to a domain name to separate sections (e.g., blog.mysite.com). These can often be created for free from your main domain.
target="_blank"
The HTML attribute used on links to ensure they open in a new tab, a best practice for external links to keep users on your portfolio.
Vendor Lock-in
The major risk of using "No-Code" builders (Wix/Squarespace) where you don't own the code and cannot easily move your site to a new host.
Webhook
The automated "digital tripwire" that notifies a cloud host (Netlify) the exact second new code is pushed to a repository (GitHub).
