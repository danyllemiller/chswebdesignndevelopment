Tech Talk: Chapter 10

Exhaustive Vocabulary for Multimedia & Data Display

Alt Text (Alternative Text)
A mandatory attribute for the <img> tag that provides a textual description of an image for accessibility and SEO.
Aspect Ratio
The proportional relationship between a display's width and height (e.g., 16:9). Critical for preventing layout shift during media loading.
Autoplay
A video/audio attribute that triggers playback immediately upon page load. Modern browsers usually block this unless the muted attribute is also present.
Bandwidth
The capacity of a network connection to transfer data. High-resolution, unoptimized images consume excessive bandwidth, slowing down site performance.
Bicubic Sharper
A specific resampling algorithm in Adobe Photoshop designed to maintain detail and sharpness when reducing an image's pixel dimensions.
Colspan
An HTML table attribute used on a <td> or <th> to make a single cell stretch horizontally across multiple columns.
Compression (Lossy vs. Lossless)
Lossy: Shrinks file size by permanently discarding data (JPG).
Lossless: Shrinks file size without removing any visual data (PNG/SVG).
Controls
The Boolean attribute for <video> and <audio> that enables the native play, pause, volume, and full-screen interface for users.
Crop
The process of removing the outer parts of an image to improve framing, change the aspect ratio, or focus on a specific subject.
Favicon
Short for "Favorite Icon." A small square brand image (typically PNG or ICO) that appears in the browser tab next to the page title.
Figcaption
The semantic HTML tag used to provide a visible, descriptive caption for content contained within a <figure> element.
GIF (Graphics Interchange Format)
A legacy raster format that supports a limited color palette (256 colors) and simple, looped animations.
MAMP (Local Server)
Software used to run a local web server environment on a computer, required to load advanced features like WebVTT subtitle tracks.
Metadata (WebVTT)
"Web Video Text Tracks." A text-based file format (.vtt) used to store timed captions, subtitles, and descriptions for media.
Muted
An attribute that ensures video content plays without sound by default. This is often a browser requirement for the autoplay feature.
Optimization
The technical workflow of balancing visual quality with the smallest possible file size to ensure fast loading times across all devices.
PNG (Portable Network Graphics)
A lossless raster format that supports alpha-transparency, making it ideal for logos, icons, and graphics with irregular edges.
Poster Image
A static image specified in the <video> tag that acts as a placeholder thumbnail until the user initiates playback.
Raster vs. Vector
Raster: Pixel-based images (JPG/PNG).
Vector: Math-based graphics (SVG) that remain infinitely sharp at any size.
Rowspan
An HTML table attribute used to make a single cell stretch vertically down across multiple rows.
Srcset
An <img> attribute that provides a list of image source files for the browser to choose from based on the user's screen resolution.
SVG (Scalable Vector Graphics)
An XML-based vector format. Because it is made of code, it can be manipulated by CSS (colors/widths) and never loses quality when scaled.
Track Tag
The child element of a <video> tag used to link external timed text files for accessibility compliance.
WebP
A modern image format developed by Google that provides superior lossy and lossless compression for images on the web.
