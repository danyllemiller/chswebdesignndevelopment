Every element is a box with four layers.

Layer Location Purpose
Padding Space the border. The element's inner cushioning.

The display Property (How Elements Flow)
Value Analogy Key Behavior
display: block; The Takes up the full available width and stacks vertically (e.g., <div>, <p>).
display: inline; The Takes up only as much width as its content; ignores width/height settings.
display: inline-block; The Sits in a line but DOES respect width and height settings.

  • • transition: The property that makes style changes (professional animations).
  • • box-shadow: A property often combined with :hover to create a effect, making an element appear to "lift up."
Responsive Units
  • • rem: The unit. It is relative to the font size of the element.
  • • vh/vw: Viewport units that are based on a percentage of the browser window's or .
Media Queries
  • • Definition: CSS "" statements that apply styles only if the screen is a certain size.
  • • Mobile-First: The strategy of designing the site for screens first, and then using a media query (with min-width) to apply larger styles for desktops.
