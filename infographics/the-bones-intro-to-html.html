The HTML Architect's Skeleton

1. The Boilerplate Hierarchy

The Brain
<head> Section

Contains Invisible Metadata: Character sets, viewport rules, and the page <title>.

The Content
<body> Section

Holds everything the user sees: Headings, paragraphs, images, and semantic structure.

The Nesting Rule

First-In, Last-Out: Zip the inner bag before the outer bag. Overlapping tags break the code.

<p>Correct <em>nesting</em></p>

Semantic Blocks

  • <header>: Logo & Intro
  • <nav>: Navigation block
  • <main>: Dominant content
  • <footer>: Closing/Legal

The Anatomy of an Image Tag

<img src="file.jpg" alt="description">
  • src: Path to the file.
  • alt: Mandatory description for accessibility (a11y).
