The Modern Pipeline

Legacy FTP (FileZilla) is manual and risky. Professionals use CI/CD to automate the launch sequence.

1. LOCAL
Code & Commit
2. HUB
Push to GitHub
3. LIVE
Auto-Deploy
The Webhook Tripwire:

When you push code, GitHub sends a signal (Webhook) to Netlify. The CDN then hosts your site globally for speed.

📖

DNS Phonebook

Translating URLs to IP Addresses.

A Record: Points to numeric IP
CNAME: Points to Netlify URL
Propagation: 24hr spread time

Subdomain: A prefix (e.g., blog.) added to your main domain.

⚠️

The Path Paradox

The #1 reason sites break upon deployment.

WRONG: Absolute Path
C:/Users/Hannah/Desktop/img.jpg
(Breaks on server!)
RIGHT: Relative Path
images/logo.png
(Works globally!)

QA Checklist

The mandatory final audit before sharing.

Resize Test: Check mobile responsiveness.
Link Rot: Click every link (404 check).
Alt Audit: Accessibility check for images.
Lighthouse: Aim for Green (90+) across board.
💼

Career Portfolio

A hired developer "Shows, Don't Tells."

  • Double Links: Always link Live Site + Code.
  • PDF Resume: The standard for scanning.
  • STAR Method: Context, Task, Action, Result.
  • Favicon: The final branding touch.
Index Rule: Default home page MUST be index.html.
