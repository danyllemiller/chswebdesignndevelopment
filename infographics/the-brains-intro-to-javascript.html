How JavaScript "listens" for user actions like clicks or typing. This connects the Bones to the Brain.

target.addEventListener('click', myFunction);

The element (querySelector)

The Equality Trap

Using the wrong operator is the #1 cause of logic bugs. Precision is everything.

= (Assignment): Forces a variable to become a value. let x = 5;
== (Loose): Guesses similarity. 5 == "5" is True. Avoid this!
=== (Strict): Checks Value + Data Type. 5 === "5" is False.

Pro-Tip: Use Number() to convert strings before doing math.

DOM Modifier Guide

Commands to change your page content and appearance on the fly.

  • .innerText: Changes text inside a standard tag (h1, p).
  • .value: Gets/Sets text inside an Input box.
  • .style.color: Directly paints CSS onto an element.
  • .classList.toggle(): Turns a CSS class on/off (Show/Hide).
element.classList.toggle('active');
