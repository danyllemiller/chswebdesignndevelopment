The Canvas API & Coordinates

In standard web design, we use HTML elements like <div> and CSS rules to build grids. But if you want to build a promotional game with 500 asteroids flying across the screen, creating 500 HTML div elements will cause the user's browser to crash. The DOM is simply too heavy for that much movement.

Instead, we use the HTML5 <canvas> tag. Which HTML element is designed for high-performance drawing via scripting? <canvas>. The canvas is literally just a blank digital painting frame. We use JavaScript to grab a set of "digital paintbrushes" and manually paint our graphics directly onto it.

The 2D Context (ctx)

To start painting, we need the tools. Which method is used to get the 2D drawing context for a canvas element? getContext('2d'). We usually store this in a variable named ctx.

The Coordinate System (Not Like Math Class!)

In your geometry class, the origin point (0,0) is in the center, and Y goes UP. A computer screen works differently.

  • Which coordinate represents the top-left corner of the canvas? (0, 0).
  • If you want to move an object to the right, what must you do to its X coordinate? Add a positive value to X. (X increases left to right).
  • If you increase an object's "Y" coordinate value, which way does it move on the screen? It moves DOWN! Y increases from the top to the bottom. If you want a bullet to travel upwards, what should its Y-velocity be? A negative number.
Drawing Shapes & Images

Once we have ctx, we can draw. Which JavaScript method is used to draw a solid, colored rectangle? fillRect(x, y, width, height).

But what if we want it to be red? Which Canvas property is used to set the color of a shape before it is drawn? ctx.fillStyle = "red";.

Pro Tip:

Z-Index in Canvas: What is 'Z-Index' equivalent in Canvas? It is simply The order in which you draw. There is no CSS `z-index` inside the canvas. Things drawn *later* in your JavaScript file will appear 'on top' of things drawn earlier.

Rotation: How do you rotate an object on the canvas? You must use ctx.rotate() after translating to the object's center, otherwise it will swing in a giant circle around the top-left (0,0) origin point!

Animation & The Game Loop

How does a flipbook work? You draw an image, turn to a blank white page, and draw the image shifted slightly. Computer animation is the exact same. We must completely wipe the canvas clean before drawing the next frame.

The Pro Engine: requestAnimationFrame

Which method tells the browser to perform an animation and requests a call to a function to update before the next repaint? requestAnimationFrame(). Why is requestAnimationFrame better than setInterval for games? Because it syncs with the monitor's refresh rate (for smooth 60fps motion) and automatically pauses when the user switches tabs, saving battery life. This is critical for meeting our agency's WRS Reliability standards.

How do you stop a game loop when the player loses? You use cancelAnimationFrame().

The 3 Steps of a Game Loop

When setting up a game loop, in what order should the core functions generally execute? Clear canvas -> Update logic -> Draw objects. This happens 60 times a second.

  • 1. Clear: How can you completely clear the entire canvas element ctx? You use ctx.clearRect(0, 0, canvas.width, canvas.height). What happens if you don't call clearRect() or fillRect() at the start of every frame? The previous frames' drawings will stay on the screen, creating an ugly smearing effect!
  • 2. Update: The "Update" portion of the game loop is responsible for calculating new positions and checking collisions. (e.g. player.x += speed;).
  • 3. Draw: Finally, you render the new graphics.

Walkthrough: The Game Loop 

let playerX = 50;
let speed = 5;

function gameLoop() {
    // 1. CLEAR
    ctx.clearRect(0, 0, canvas.width, canvas.height);
    
    // 2. UPDATE
    playerX += speed; // Move right
    
    // 3. DRAW
    ctx.fillStyle = "blue";
    ctx.fillRect(playerX, 50, 40, 40);
    
    // LOOP
    requestAnimationFrame(gameLoop);
}
gameLoop(); // Start the engine

Pro Tip:

Saving State: What does the ctx.save() and ctx.restore() pair do? It saves and resets the current drawing state (rotation, scale, colors). Think of it like taking a snapshot of your settings so you can mess with them for one specific object, and then go back to normal for the rest.

High Refresh Rates: If a game runs too fast on high-refresh-rate monitors (like 144Hz gaming screens), what concept is used to normalize the speed? Delta Time. This tracks the difference in time between the current frame and the last frame to ensure a player moving 10px per second actually moves 10px, regardless of framerate.

OOP & Classes (The Blueprint)

If you are building a game with 50 aliens, it would be a nightmare to create 50 `alienX` and 50 `alienY` variables. This is "Spaghetti Code," and it wastes our agency's billable hours. Instead, we use Object-Oriented Programming (OOP).

What is the name of the "Blueprint" used in Object-Oriented Programming to create multiple objects? A Class. What is the main advantage of using a Class to build game objects? It encapsulates properties and methods for reuse. You write the code once, and you can spawn a thousand enemies perfectly.

Anatomy of a Class
  • Naming: What naming convention is the industry standard for Class names? Capitalized (PascalCase), like `class Enemy`.
  • Constructor: In JavaScript ES6, what function runs automatically when a new instance of a class is created? The constructor(). This is where you set the starting X/Y coordinates.
  • Context (this): In a class, what does the 'this' keyword refer to? The specific instance of the object being created/updated. If you forget to include `this.` before a variable inside a class method, what happens? The computer looks for a global variable instead, causing a Reference Error. Which property of a class instance allows it to 'know' how much health it has left? A property defined in the constructor, like this.health.
  • Methods: How do you define a new method inside a JavaScript class? You just write MethodName() { } without using the word `function`.
Instantiating & Arrays

How do you create a brand new enemy from an Enemy class? You use the `new` keyword: new Enemy(). When spawning many, we store them in an array.

  • Which array method is best for adding a newly spawned projectile to a list of active bullets? bullets.push().
  • If you have an array of asteroids, how do you remove the first one from the array after it is destroyed? You use asteroids.shift().
  • If you use asteroids.filter(a => !a.destroyed), what are you doing? You are creating a new array containing only asteroids that are NOT destroyed. This is how we clean up memory!

Walkthrough: The Class Blueprint 

class Particle {
    constructor() {
        this.x = Math.floor(Math.random() * canvas.width);
        this.y = 0; // Starts at top
        this.speed = Math.random() * 3 + 1;
    }
    update() {
        this.y += this.speed; // Falls down
    }
    draw() {
        ctx.fillStyle = "white";
        ctx.fillRect(this.x, this.y, 5, 5);
    }
}
// Spawn 10 particles into an array
const particles = [];
for(let i=0; i<10; i++) {
    particles.push(new Particle());
}

Physics, Collisions, & User Input

How does a video game actually 'know' when a bullet hits a monster? A "Hitbox" is used in game logic to determine if two objects are overlapping or touching. What is a 'Hitbox'? It is The invisible area used to calculate collisions, which is often smaller than the actual graphic being drawn.

AABB Collision (Rectangles)

AABB Collision detection stands for Axis-Aligned Bounding Box. It requires checking the overlap of which properties? x, y, width, and height. Which operator allows us to check multiple collision conditions at the same time? We MUST use the && (AND) operator, because all four sides must overlap simultaneously for a true hit.

// The Pro AABB Formula
if (rect1.x < rect2.x + rect2.width &&
    rect1.x + rect1.width > rect2.x &&
    rect1.y < rect2.y + rect2.height &&
    rect1.y + rect1.height > rect2.y) {
    // Collision detected!
}
Circular Collision

For circular hitboxes, we don't use X and Width. To check for circular collision, which mathematical theorem is typically applied to find the distance between center points? The Pythagorean theorem ($a^2+b^2=c^2$). In circular collision detection, when has a collision occurred? When the distance between centers is less than the sum of the radii.

Physics Math
  • Canvas Boundaries: To prevent an object named player from moving off the left edge of the canvas, which condition should you check? if (player.x < 0). If player.y + player.height >= canvas.height evaluates to true, what is happening? The player is touching the bottom of the canvas.
  • Bouncing: How do you make an object 'bounce' off the bottom of the canvas? You reverse its direction: velocity.y *= -1.
  • Screen Wrapping: How do you create an 'infinite' wrap-around effect on the X-axis? if (x > width) x = 0.
  • Speed: To make an object move faster, you should Increase the value added to its position per frame.
  • Gravity: To apply basic gravity to a jumping character, what must you do every frame? Add a constant value to the character's vertical velocity. Simulating gravity usually involves adding a small value to an object's Y-velocity.
  • Friction: What is 'Friction' in game physics? Gradually reducing velocity over time (e.g. `speed *= 0.98`).
Keyboard & Mouse Inputs

Which event listener is typically used for continuous character movement in games? The keydown event. How do you detect if a user has released a key to stop moving? Listen for the 'keyup' event.

What property of a keyboard event tells you the physical key that was pressed? event.code (e.g. 'ArrowRight' or 'Space'). How do you check if a specific key (like 'Space') is currently being held down? Store the key state in a boolean variable inside keydown/keyup listeners.

Which event listener is used to track the mouse position on the canvas? 'mousemove'.

Advanced Game Logic & Graphics

State Management

What is the purpose of a 'Game State' variable (like gameState = 'menu')? To control which screens or logic blocks are active. A Game State variable is used to control which logic runs at different times.

When building a Start Screen, how do you prevent the main game loop from running until a button is clicked? Wrap requestAnimationFrame inside a conditional state check (e.g. `if(gameState === 'playing') requestAnimationFrame(loop);`).

Advanced Graphics
  • Sprite Sheets: What is 'Sprite Sheet' animation? One large image containing multiple frames of animation. This is much faster for the browser to load than 20 separate image files!
  • Parallax: What is 'Parallax Scrolling'? Background layers moving at different speeds to create depth.
  • Transparency: What does ctx.globalAlpha control? Transparency of everything drawn on the canvas.
  • Particles: What is the primary purpose of 'Particle Systems' in games? To create effects like fire, smoke, or explosions using many small objects.

Need to center something exactly? What is the result of canvas.width / 2? The horizontal center of the canvas.
