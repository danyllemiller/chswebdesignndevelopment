III. Typography, Sizing, and The Box Model
Sizing Units
  • • Absolute Unit: A fixed size that never changes (e.g., the unit).
  • • Relative Unit (MVP): (rem) is relative to the font size of the element, making it the best unit for scalable design.
  • • Viewport Unit: Units like (vw) are based on the width or height of the user's browser window.
Typography
  • • Font Stack: A prioritized list of fonts (e.g., "Roboto", Arial, sans-serif) that ensures the browser always has a option.
  • • Generic Font Families: (fonts with "feet," like Times New Roman) vs. (clean fonts, like Arial).
  • • @font-face: The special CSS rule that allows you to upload and use font files.
The Box Model

The most important concept in CSS: every element is a box with four layers.

Box Model Layer Description What it does
Content Your actual text or image.
Padding The space inside the box. Adds cushioning between the content and the border.
Border The line around the outside of the padding.
Margin The element's "personal space bubble." Adds space outside the box, pushing other elements away.
