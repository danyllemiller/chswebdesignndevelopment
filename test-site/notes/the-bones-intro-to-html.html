PART 2: What is an HTML File? (The Boilerplate)

Every HTML file must contain the core structure:

  • DOCTYPE: The declaration, which is written as for HTML5.
  • <head>: The of the page; contains (data the user doesn't see).
  • <body>: Contains the user does see (text, images, links).

PART 3: Tags, Elements, and Attributes
  • Element: The entire unit: tag + content + tag.
  • Empty Element: A tag that does not have a closing tag (e.g., or ).
  • Nesting Rule: The tag you open is the one you must close.
  • Attribute: Special information added to the tag to give the element extra instructions (e.g., id or class).
