TL;DR: Chapter 14

CMS Strategy, WordPress & Maintenance Cheat Sheet

The Manager's Code of Conduct

1. Choose Your Ownership Model
  • WordPress.com (Rental): Easy but limited. Host manages everything. You follow their rules.
  • WordPress.org (Owner): Professional standard. Open-source software you host yourself for total freedom and ownership.
2. The "Clothes" vs. The "Apps"
  • Theme: Controls visual design. Swapping themes changes the "Clothes" but leaves the content (Database) intact.
  • Plugin: Mini-apps (like WooCommerce) that add specific powers like stores or contact forms.
3. Timely vs. Timeless Content
  • Posts: Timely news stories. Reverse-chronological. Use Categories/Tags for search.
  • Pages: Timeless navigation items (About Us). Found in Appearance > Menus.
4. Publishing & Admin

Enter the Dashboard via /wp-admin. Use the Block Editor (Gutenberg) to rearrange content with the '+' icon. Use Revisions to undo mistakes.

5. Pro Strategy: SEO & Security

Optimize Permalinks for readable URLs. Use Caching for speed. Assign Roles (Administrator vs. Editor) to protect settings.

The Golden Rule of Maintenance
Before updating the WordPress core, a theme, or a major plugin, you must perform a full database and file Backup.
Beware of Vendor Lock-in
No-Code builders (Wix/Squarespace) are easy for beginners but prevent you from moving your code to a different host later.
