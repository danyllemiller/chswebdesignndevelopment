TL;DR: Chapter 3

1. Intellectual Property (IP)

Copyright: Automatic creative shield. Trademark: Brand identity (Nike). Patent: Invention protection (lasts 20 years). Trade Secret: Indefinite protection for code logic.

2. Security (HTTPS & SSL)

HTTP is a Postcard. HTTPS is a Sealed Letter. It uses an 8-step handshake and an SSL/TLS certificate to encrypt data.

3. Privacy Laws

GDPR: EU law, Opt-In model, Right to be Forgotten. SB220: Nevada law, Opt-Out model for data sales.

4. Accessibility (a11y)

Follow POUR principles. target: WCAG Level AA. Audit for Alt Text, Color Contrast (4.5:1), and Keyboard access. It's an ADA civil right.

ETHICS ALERT: Avoid Dark Patterns (tricking users) and Algorithmic Bias (prejudice in AI datasets).
