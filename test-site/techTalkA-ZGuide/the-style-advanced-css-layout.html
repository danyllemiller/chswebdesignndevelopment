Tech Talk: Chapter 9

Advanced Vocabulary for Professional Architects

Vocabulary Expansion

clamp()

A CSS function that clamps a value between an upper and lower bound. Used for Fluid Typography to make text grow and shrink smoothly with the screen size.

CSS Variables (Custom Properties)

Entities defined by CSS authors that contain specific values to be reused throughout a document. Declared in the :root and accessed via the var() function.

minmax()

A CSS Grid function that defines a size range greater than or equal to min and less than or equal to max. Essential for preventing grid items from getting too small.

Overflow

The CSS property that controls what happens when content is too big for its box. Options include visible, hidden, scroll, and auto.

Stacking Context (z-index)

The three-dimensional conceptualization of HTML elements along an imaginary Z-axis. z-index determines which overlapping element appears "on top."

Sticky Positioning

A positioning type that toggles between relative and fixed, depending on the user's scroll position. Useful for headers that stay visible only when needed.

Test Alert!

Q: Difference between auto-fill and auto-fit?

A: auto-fill keeps empty tracks; auto-fit collapses empty tracks to stretch existing items.
