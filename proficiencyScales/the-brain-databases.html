Learning Targets

  • Technical: I can differentiate between Relational (SQL) and Non-Relational (NoSQL) databases and choose the correct structure for a given dataset. (NV 5.4.1)
  • Technical: I can explain the CRUD lifecycle (Create, Read, Update, Delete) and its application in data management. (NV 5.4.1)
  • Technical: I can write basic SQL queries using commands such as SELECT, FROM, WHERE, INSERT, and ORDER BY. (NV 5.4.1)
  • Technical: I can define primary keys and relationships between tables to ensure data integrity. (NV 5.4.1)
  • Employability: I can demonstrate analytical reasoning and cognitive flexibility to design efficient database schemas that minimize data redundancy. (WRS 1.2.3)
  • Employability: I can demonstrate probity and professional decorum by identifying the ethical implications of data storage and privacy laws (e.g., GDPR/CCPA). (WRS 1.1.2, 1.1.4)
