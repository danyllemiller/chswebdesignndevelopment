The TL;DR: What You Really Need to Know

The Core Concepts of CSS

  • CSS makes HTML look good. Write styles in an external .css file .
  • The most specific selector wins: ID ( #main ) > Class ( .cool-thing ) > Element ( p ) .
  • Everything is a box. Learn the Box Model (margin, border, padding) to control spacing.
  • Use rem for font sizes. It's flexible and scalable.
  • Flexbox ( display: flex ) is for 1D layout (rows or columns). It's awesome for centering.
  • Grid ( display: grid ) is for 2D layout (rows and columns). It's for whole-page designs.
  • position: fixed sticks an element to the screen. position: absolute takes it out of the flow.
  • Use the W3C CSS Validator to find bugs in your code.
