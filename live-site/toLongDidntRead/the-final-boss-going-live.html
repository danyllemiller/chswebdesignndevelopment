TL;DR: Chapter 18

Deployment, DNS & Portfolio Strategy

Launch Rules

1. FTP vs. CI/CD
  • FTP (File Transfer Protocol): The legacy "Moving Truck." Manual dragging of files (FileZilla). High risk of overwriting live files.
  • CI/CD (Continuous Deployment): The modern "Robot." Connects GitHub to a host (Netlify). When you push, the site updates automatically via Webhooks.
2. The Path Rules

Absolute Path: C:/Users/Hannah... (ONLY works on your laptop; breaks on launch).
Relative Path: images/logo.png (Professional standard; works everywhere).

3. DNS (Internet Phonebook)

A Record: Points name to numeric IP Address.
CNAME: Points name to another URL (like Netlify).
Propagation: The 24-hour wait for changes to reach the global network.

4. Professional QA

Use Google Lighthouse to grade your site (0-100). Check Console for red errors. Perform an Alt Audit for screen readers. External links MUST use target="_blank".

The Index Rule
Your homepage MUST be named index.html. Web servers look for this specific file first by default.
