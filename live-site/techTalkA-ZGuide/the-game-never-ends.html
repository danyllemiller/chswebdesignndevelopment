Tech Talk: Chapter 17

Exhaustive Vocabulary for Emerging Tech & Web 3.0

AI (Artificial Intelligence)
The broad concept of machines acting smartly or simulating human intelligence to perform tasks.
Algorithmic Bias
When an AI system produces systematically prejudiced results due to flawed or non-diverse training data.
API (Application Programming Interface)
The "Waiter" or contract that allows two different pieces of software to talk to each other and exchange data.
API Key
A secret digital ID badge used to identify and authenticate a program when it calls an API service.
AR (Augmented Reality)
Technology that adds digital elements (graphics or sounds) onto a live view of the physical world.
Async / Await
Modern JavaScript syntax used to pause code execution until a background network request (Promise) is finished.
Auth Token
A temporary digital "VIP Pass" used to prove a user's identity during a secure session.
CDN (Content Delivery Network)
A global network of servers that hosts copies of website assets closer to the user to reduce load times.
Chatbot
An AI-driven interface designed to simulate human conversation, often used in customer service.
Client
The "Customer" in the network model; usually the user's web browser or device requesting data.
Cloud Computing
The practice of renting computer power (servers, storage) from a remote provider over the internet.
Component
A reusable UI building block (like a button or navigation bar) used in frameworks like React to build apps faster.
Data Center
A massive physical building filled with powerful servers that host "The Cloud."
DevOps Engineer
A professional who ensures servers stay online and that code deploys smoothly from GitHub to the live web.
Endpoint
The specific URL address where an API "listens" for a request (e.g., api.site.com/v1/users).
Encryption (SSL/TLS)
The process of scrambling data into a secret code so it cannot be read by hackers during transit.
Framework (React / Vue)
A pre-written "LEGO kit" of code and components used by teams to build complex web applications efficiently.
Full Stack Developer
A developer who works on both the Front-End (user interface) and the Back-End (databases and servers).
GET Method
The standard HTTP method used by a client to "retrieve" or read data from an API server.
Haptic Feedback
Communication through physical touch or vibration, common in smartwatches and game controllers.
Headless UI
Interfaces designed for devices without traditional screens (like Alexa or smart speakers).
HTTP Status Codes
The server's response code: 200 (Success), 404 (Not Found), 500 (Server Error).
IoT (Internet of Things)
A network of physical hardware (smart fridges, streetlights) connected to the internet.
JSON
JavaScript Object Notation; the standard lightweight key-value format used for API data delivery.
Machine Learning (ML)
A subset of AI where systems improve their performance based on exposure to data rather than hard-coding.
MFA (Multi-Factor Auth)
A security system requiring two or more proofs of identity (e.g., a password and a text code).
MPA (Multi-Page App)
A traditional website that refreshes the browser and loads a new HTML file on every click.
Pattern Recognition
The AI logic used to filter spam email, recognize faces, or predict user habits.
Payload / Body
The actual data being sent to a server during a POST or PUT request.
Promise
A JS object representing the eventual completion (or failure) of a background network task.
Prototype
An interactive, clickable model of an app used for testing before the final code is written.
Sensors
The physical hardware components in IoT devices that collect environmental data.
Serverless Computing
When developers upload functions that run on demand without managing the physical server hardware.
SPA (Single Page App)
An app that loads one HTML shell and updates content instantly via JavaScript without refreshing.
VUI (Voice User Interface)
An interface designed specifically for natural language and speech interaction.
Web 3.0
The next era of the web focused on decentralization, blockchain, and AI connectivity.
Webhook
An automated digital tripwire message sent between apps to trigger events instantly.
Wireframe
The low-fidelity visual blueprint used to plan an app's layout and logic before coding begins.
