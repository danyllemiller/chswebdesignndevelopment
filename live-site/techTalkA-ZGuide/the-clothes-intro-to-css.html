Essential Terminology

Absolute units
Fixed units of measurement that don't change, like pixels ( px ).
Attribute Selector
A selector that targets an element based on its HTML attributes, like [type="submit"] .
Browser styles
The default styles that a web browser applies to all websites. Also called user agent styles.
CSS (Cascading Style Sheets)
The language used to style the visual presentation of a website.
CSS at-rule
A special CSS instruction that starts with an @ symbol, like @charset or @import .
Embedded Styles
CSS rules placed inside a <style> tag in the HTML file's <head> . (Also: Internal Styles).
External Styles
CSS rules placed in a separate .css file, which is the recommended method.
Flexbox
A CSS layout module ( display: flex ) for arranging items in a single dimension (a row or a column).
Grid
A CSS layout module ( display: grid ) for arranging items in two dimensions (rows and columns).
HSL
A way to define color based on Hue, Saturation, and Lightness.
Inline Styles
A style rule applied directly to an HTML element using the style attribute.
Position
A CSS property ( static , relative , absolute , fixed ) that controls how an element is placed on the page.
Progressive enhancement
A design strategy where you build a baseline experience, then add advanced features for capable browsers.
pseudo-class
A keyword added to a selector to style an element in a special state (e.g., :hover ).
pseudo-element
A keyword added to a selector to style a specific part of an element (e.g., ::first-letter ).
rem or root em unit
A relative unit that is based on the font size of the root ( <html> ) element.
Relative units
Flexible units of measurement that are based on another value, like rem or % .
RGB triplet
A method of defining color by its Red, Green, Blue light values.
selector patterns
The different ways you can target HTML elements with CSS.
style inheritance
The process where some CSS properties on a parent element are passed down to its children.
style rule
A single CSS instruction, made of a selector and a declaration block.
Typography
The art and technique of arranging text to make it readable and appealing.
Validation (CSS)
The process of checking CSS code against official standards.
vendor prefix
A prefix added to a CSS property (like -webkit- ) for browsers to test experimental features.
viewport unit
A relative unit based on the size of the browser's viewport, like vw (viewport width).
web font
A font file that is downloaded by the browser so it can be used on a website.
Wildcard
The * selector, which selects all elements.
