CHS Web Design Track

Web Design I (Level 1A & 1B)

Building the Foundations | Year 1

Course Description

Semester 1 (Foundations) focuses on the professional "design-first" process. Students learn career professionalism (SkillsUSA), web ethics, and law. They analyze audiences and create sitemaps/wireframes before writing HTML5.

Semester 2 (Front-End) brings sites to life. Students master CSS (Box Model, Flexbox, Grid) and responsive layouts. They learn to embed media and ensure ADA compliance, finishing with a JavaScript-powered portfolio.

Key Learning Units

1 The Developer's World

History of the web, W3C standards, and the Guild mindset.

2 The Project Blueprint

ADDIE Cycle, User Personas, SEO, and Wireframing.

3 Structural Bones

HTML5 Semantic markup, lists, and link architecture.

4 Visual Presentation

Responsive CSS, Flexbox, and CSS Grid Mastery.

Course Outcomes
  • ✔️ Design multi-page websites
  • ✔️ Write/Debug HTML5 & CSS3
  • ✔️ Remediate ADA/A11y issues
  • ✔️ Basic DOM Manipulation (JS)
