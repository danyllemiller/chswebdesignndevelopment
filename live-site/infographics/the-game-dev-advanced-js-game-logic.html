The 60 FPS Loop

Professional games use requestAnimationFrame() because it syncs with the monitor and pauses automatically when inactive.

Mandatory Execution Order (C.U.D.):
1. CLEAR
ctx.clearRect()
2. UPDATE
Physics, Collisions, Movement
3. DRAW
ctx.drawImage()

Bugs: Forgetting 'Clear' causes object "smearing" (ghosting).

Controlling Chaos

We use the Math Object to spawn items and normalize speeds across hardware.

Math.floor(Math.random() * 10)

Returns an integer between 0 and 9. It will never reach 10!

Delta Time:

Uses time since last frame to ensure speed is consistent on slow vs fast computers.

Developer Toolbox

Management methods for smooth interaction.

  • 'keydown' / 'keyup': Listen for start/stop movement.
  • event.code: Detect physical keys (e.g., 'Space').
  • .shift(): Remove first item from an array (destroyed enemies).
  • .splice(): Remove item from middle of array.
  • Haptic Feedback: Vibration alerts for Gabe.
