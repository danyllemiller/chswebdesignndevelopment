Single Page Apps (SPAs)

Websites are becoming software. "Chloe" expects speed, and SPAs deliver by eliminating page reloads.

MPA
Refreshes every click.
SPA
JS swaps content instantly.

Frameworks: Professionals use React or Vue to build with reusable Components.

🛎️

The API Waiter

How apps talk to each other. Your website is the "Customer," and the API is the messenger.

1. The GET Request: Client asks for data using a specific Endpoint URL.
2. Authentication: Your API Key acts as your digital ID badge.
3. JSON Delivery: Data is served in a lightweight key-value text format.

Codes: 200 (OK), 404 (Missing), 500 (Server Error).

🏠

Internet of Things (IoT)

The physical web. Hardware devices connected to the network using Sensors to collect data.

  • Smart Cities: Streetlights that sense pedestrians.
  • VUI: Designing for Voice interfaces (Alexa/Siri).
  • Haptics: Communication through physical vibration.
SECURITY RISK:

Mass-produced IoT devices often ship with weak default passwords, creating easy backdoors for hackers.

Digital Footprint

The permanent record of a user's activity. Protecting this from Invasion of Privacy is the developer's duty.

Authentication

Modern security requires MFA (Multi-Factor Auth)—proof using a password AND a second device.
