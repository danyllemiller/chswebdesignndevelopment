TL;DR: The "Game Dev"

1. The Engine (The Loop)

Modern games run on a continuous loop powered by requestAnimationFrame(). This syncs with the screen's refresh rate and pauses automatically when inactive. Professional order: Clear Canvas → Update Logic → Draw Objects.

2. Blueprints (OOP)

To spawn dozens of enemies, use ES6 Classes. This encapsulates properties (speed, health) and methods (update, draw). The constructor() runs once when an object is created via the new keyword. Use this to modify that specific object.

3. Boundaries & Collision

Check if a player hits a wall: if (player.x + player.width >= canvas.width). Collision (AABB) checks for the overlap of x, y, width, and height. Circles require the Pythagorean Theorem to find distance between centers.

4. Physics & Chaos

Use Math.floor(Math.random() * 10) to get an integer between 0 and 9. Simulate Gravity by adding a constant value to vertical velocity every single frame. Use Delta Time to keep speed consistent across different computers.

Exam Tip: Use keyup to stop movement when a key is released. Use shift() to remove the first destroyed element from an array.
