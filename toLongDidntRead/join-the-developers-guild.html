TL;DR: Chapter 1

1. Internet vs. Web

The Internet is the physical hardware. The Web is the software service running on it via the Client-Server model. An Intranet is a private network.

2. The Address Book (DNS)

Computers use IP Addresses (numbers). Humans use Domain Names (like google.com). The DNS is the phonebook translating names to numbers, governed by ICANN.

3. URLs & Security

A URL has a Protocol, Domain, TLD, and Path. HTTPS encrypts data in transit using an SSL Certificate and public/private keys.

4. Hosting & CDNs

Hosting is renting server space (Shared, VPS, Dedicated, Cloud). CDNs (Content Delivery Networks) store copies globally to speed up your site.

5. Languages & UI/UX

HTML: Bones. CSS: Clothes. JavaScript: Brains. Good design relies on Usability, White Space, and Visual Hierarchy to guide the user.

6. Careers & SEO

Tech roles include Front-End, Back-End, and UI/UX. Sites use SEO to rank on search engines. Avoid deceptive spamdexing.

7. The Guild & Soft Skills

Communication and Rubber Duck Debugging are vital. Join a CTSO (SkillsUSA/FBLA). The Agile Triangle dictates: Fast, Cheap, Good (pick two).

8. Law & Ethics

Copyright is automatic; use Creative Commons legally. Avoid "Dark Patterns". Embrace the "Curb Cut Effect" for Accessibility. Protect secrets with an NDA.
