TL;DR: Chapter 10

Media Optimization & Embedding Cheat Sheet

The Core "Rules" of Media

1. Choose the Correct Format
  • Photos: Use JPG (high detail, small size) or WebP (modern best compression).
  • Logos/Icons: Use SVG (infinitely sharp code) or PNG (if transparency is needed).
2. The Optimization Funnel

NEVER upload a raw photo. Use Photoshop to: Crop (Focus) → Resize (Shrink dimensions to 800px-1200px) → Compress (Save for Web at 60-80% quality).

3. Video Best Practices
  • iframe: Best for YouTube/Vimeo. Use aspect-ratio: 16 / 9 in CSS to keep it responsive.
  • Native <video>: Use for local files. Always include controls and <track> for accessibility.
4. Semantic Tables

Tables are for data, not layout. Use <thead>, <tbody>, and <tfoot>. Use colspan to merge columns for summaries.

Manager's Golden Rule
Always include alt text on images and <track> subtitles for video. Accessibility is not optional—it's the law.
The MAMP Trick
Subtitles (.vtt files) will not load if you just open the file from your folder. You must use localhost (MAMP) to see them!
